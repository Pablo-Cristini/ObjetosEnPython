Nombre: Delegacion de trabajos

Este proyecto simula la delegación de tareas a empleados disponibles dentro de un supermercado. A cada empleado se le asigna un trabajo de acuerdo a la categoría que le corresponde. Cada trabajo tiene una cantidad de horas asignadas y un valor por hora. No es objetivo del proyecto calcular sueldos ni liquidaciones, sino automatizar la asignación de tareas de forma ordenada y eficiente.

El código se utiliza creando manualmente los trabajos existentes, discriminados por categoría. También deben instanciarse los empleados disponibles. Una vez definidos los trabajos y empleados, el sistema se encarga de repartir automáticamente las tareas, asignándolas a quienes estén libres y pertenezcan a la categoría correspondiente. 

El programa avisa si un empleado queda sin tareas asignadas o si algún trabajo no pudo ser delegado por falta de personal adecuado. Todo este proceso ocurre dentro del archivo principal main.py, y los resultados se muestran por consola. 
Ademas se realizan validaciones ya que en los trabajos el sueldo minimo no puede ser menor a 1000 la hora y la cantidad de horas por puesto no deben ser menor a 4.

El código está desarrollado completamente en Python. El proyecto está compuesto por dos archivos principales: main.py y Clases.py. En él se aplican diversos conceptos fundamentales de la programación orientada a objetos, como herencia, composición, polimorfismo, abstracción y encapsulamiento.
